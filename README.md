# Team Task Manager

A full-stack task management web application built using Flask.

## Features
- User Signup/Login
- Role Based Access (Admin/Member)
- Project Management
- Task Creation and Assignment
- Dashboard Analytics

## Tech Stack
- Flask
- SQLite
- HTML
- CSS

## Run Locally
```bash
python app.py
```
